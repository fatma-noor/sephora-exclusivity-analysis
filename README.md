# sephora_exclusivity_analysis

Exploratory data analysis of Sephora product and review data to evaluate whether Sephora-exclusive products show different engagement patterns.

- **Notebook**  
  `sephora_exclusivity_analysis.ipynb`  
  Main analysis notebook. Includes data acquisition, cleaning, exploratory visualisation, regression analysis, sentiment analysis, and clustering.

- **Scripts**  
  `styles.py`  
  Helper functions for notebook styling and formatting.

- **Data**  
  Data is not stored in this repository.  
  The notebook contains code to download the dataset directly from Kaggle using the Kaggle API.  
  Users must provide their own `KAGGLE_USERNAME` and `KAGGLE_KEY` credentials to authenticate and download the data.

- **Output**  
  All figures, tables, and model results are generated within the notebook.
